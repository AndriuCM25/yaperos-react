# Ya-pe App

Aplicación ficticia de billetera digital inspirada en aplicaciones de pagos móviles.

## Instalación

1. Clona o descarga este proyecto
2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm start
```

4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador

## Características

- ✅ Pantalla principal con saldo
- ✅ Envío de dinero simulado
- ✅ Pantalla de confirmación con animación de confeti
- ✅ Historial de transacciones
- ✅ Diseño responsive

## Tecnologías

- React 18
- Tailwind CSS
- Lucide React (iconos)
